#---Training_tracker_V2---

Бот для отслеживания тренировок

Отмечай тренировки и смотри статистику.
Бот каждый день отправляет сообщение - отметь тренировку (или ее отсутствие)при помощи инлайн клавиатуры.

Мотивацию поднять!

Версия для распространения в Docker контейнере. 
Настройка окружения - Ansible 
Обновление при помощи Git Actions - пересборка контейнера, 
запуск нового контейнера на сервере.



Для развертывания приложения на сервере - Внести данные о адресе хоста, пользователе и ключе в параметрах Ansible и запустить плейбук.
Будет выполнена настройка сервера и установка Docker.
Для запуска бота - скопировать compose.yml на сервер и создать файл .env с токеном бота.
Запустить Docker Compose.

При коммите в main происходит сборка и отправка образа в DockerHub, обновление и рестарт контейнера на сервере.
(Необходимо в секреты добавить адрес хостаб пользователя и секретный ключ)
